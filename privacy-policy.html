Privacy Policy

Last updated: April 2026  ·  Effective immediately upon installation

Plain-English Summary TabZen stores everything locally on your device. Your API key never leaves Chrome's local storage. The only external call made is to Anthropic's API (api.anthropic.com) to group your tabs — and only tab titles and URLs are sent, never page content or personal data. We collect nothing, sell nothing, and have no servers.

1. Who we are

TabZen is a Chrome browser extension that helps users organise tabs using AI, detect duplicate tabs, surface stale tabs, and manage bookmark health. It is an independent project and is not affiliated with Google or Anthropic.

For questions about this policy, contact: tabzen.privacy@gmail.com

2. What data TabZen accesses

DataPurposeStored where
Tab titles & URLs Sent to Anthropic API to generate topic-based groupings Anthropic API only
Tab last-accessed timestamps Used to detect stale tabs Local only
Bookmark titles, URLs, click counts Used to surface rarely-used bookmarks Local only
Anthropic API key (entered by user) Used to authenticate requests to Anthropic API Local only
User preferences (thresholds, grouping mode) Used to personalise extension behaviour Local only
Page content, cookies, passwords, form data Not accessed Never collected

3. How your data is used

Tab titles and URLs are the only data sent externally. They are sent to the Anthropic API (api.anthropic.com) solely to generate a grouping suggestion when you click "Run AI Grouping". This data is governed by Anthropic's Privacy Policy.

All other data — timestamps, bookmark metadata, settings, and your API key — remains entirely within Chrome's chrome.storage.local on your device and is never transmitted anywhere.

4. Data we do NOT collect

  • We do not collect any analytics or usage statistics
  • We do not read page content, cookies, passwords, or form data
  • We do not transmit your browsing history to any server
  • We do not use advertising networks or tracking pixels
  • We do not share, sell, or license any user data to any third party
  • We have no backend server — there is no TabZen database

5. Third-party services

TabZen makes API calls to Anthropic (anthropic.com) when the user initiates AI grouping. The data sent is limited to tab titles and URLs. Please review Anthropic's Privacy Policy for how they handle API request data.

No other third-party services are used.

6. Data retention & deletion

All data stored by TabZen lives in chrome.storage.local on your own device. You can delete it at any time by:

  • Uninstalling the TabZen extension (clears all stored data)
  • Clearing the extension's storage via Chrome's developer tools
  • Using the "Clear API Key" button in the extension, which removes your key from storage

TabZen has no server-side data to delete because no data is ever sent to our servers (we have none).

7. Your API key security

Your Anthropic API key is stored using Chrome's chrome.storage.local API, which is sandboxed to the TabZen extension and inaccessible to websites or other extensions. It is transmitted only to api.anthropic.com over HTTPS when you trigger an AI grouping. It is never logged, cached externally, or transmitted to any other endpoint.

8. Children's privacy

TabZen is not directed at children under the age of 13. We do not knowingly collect any data from children.

9. Changes to this policy

If this policy changes, the updated version will be posted at this URL with a revised "last updated" date. Continued use of the extension after any changes constitutes acceptance of the updated policy.

10. Contact

For any privacy-related questions or requests, contact us at: tabzen.privacy@gmail.com
